CSS-Einheiten

Absolute Einheiten

Pixel (px)

Absolut bedeutet, dass die Schriftgröße konstant bleibt, auch wenn die Basisschriftgröße verändert wird.

20 Pixel

Relative Einheite

Relativ bedeutet, dass sich die Schrifgröße an eine bestimmte Basisgröße ausrichtet - proportional verändert

EMs

Bei den EMs bezieht sich ein Kindelement immer relativ auf sein Eleternelement.

Eleternelement (2 x Basisgröße) Kindelement (4 x Basisgröße)

REMs

Bei den REMs bezieht sich ein Kindelement immer relativ auf die Basis-Schriftgröße

Eleternelement (2 x Basisgröße) Kindelement (2 x Basisgröße)

% Prozent

z.B. die Breite eines Elements auf Basis des Elternelements

vw Viewport Width

Gibt die Größe in Bezug auf die Breite des sichtbaren Bereichs des Browser-Fenster an. Im Gegensatz zu den Prozentangaben, bezieht sich das nicht auf das Elternelement sondern immer auf die Browserbreite.

vh Viewport Height

Gibt die Größe in Bezug auf die Höhe des sichtbaren Bereichs des Browser-Fenster an. Im Gegensatz zu den Prozentangaben, bezieht sich das nicht auf das Elternelement sondern immer auf die Browserhöhe.
