Responsive Design

Darunter versteht man ein Layout/Design einer Webseite/Webanwendung, dass sich an das jeweilige Anzeigegerät anpasst.

Viewport

Der sichtbare Bereich im Browser (Content, der angezeigt wird).

Breakpoint

Punkt/Seitenbreiten ab dem sich das Design/Layout sprunghaft ändert. (z.B. anstatt 4 Spalten am Desktop, wird auf einem Smartphone nur eine Spalte angezeigt).

Voraussetzungen

  1. Viewport-Meta-Tag im Head
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  2. Media-Queries im CSS
    h1 {
    font-size: 3.5rem;
}

@media (max-width: 575px) {
    h1 {
        font-size: 2.5rem;
    }
}
